# DSA Sorting Visualizer

A simple and interactive web application built using **React + Vite** to help students understand how different sorting algorithms work.  
The visualizer displays array bars and animates each step of the algorithm, making learning more intuitive.

---

##  Features

- Generate a new random array
- Step-by-step visual animations
- Separate sections for:
  - Algorithm pseudocode
  - Simple explanation of how it works
- Clean UI layout with three panels:
  - Visualizer
  - Pseudocode
  - Explanation

---

##  Algorithms Implemented

- Bubble Sort  
- Selection Sort  
- Merge Sort  
- Quick Sort  

Each algorithm updates the array step-by-step so users can follow the sorting process clearly.
